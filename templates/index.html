For transparency, this tab summarizes what the training pipeline does in train_model.py, mirroring the Alzheimer’s notebook workflow from data loading through model selection.

Step 1

Load and Prepare Data

The dataset is loaded from Gonzales_AlzheimersDisease.csv. PatientID is removed, Diagnosis becomes the target, and all remaining clinical variables are kept as model features.

Step 2

Split Train and Test Sets

The pipeline performs an 80/20 train-test split with random_state=42 to keep evaluation consistent with the notebook implementation.

Step 3

Scale Features

Numeric inputs are normalized with MinMaxScaler so models compare features on a shared range before classification.

Step 4

Train 5 Baseline Models

Logistic Regression, K-Nearest Neighbors, Decision Tree, Random Forest, and Naive Bayes are first trained as baseline classifiers.

Step 5

Handle Class Imbalance

SMOTE is inserted into the training pipeline so the minority Alzheimer’s class is oversampled during model evaluation.

Step 6

Run 5-Fold Cross-Validation

Each SMOTE-enabled pipeline is scored using stratified 5-fold cross-validation on accuracy, precision, recall, and F1 score.

Step 7

Tune Top Models

Logistic Regression and Random Forest are selected for GridSearchCV hyperparameter tuning because they emerged as the strongest candidates.

Step 8

Select the Deployed Winner

The tuned model with the better test-set F1 score is saved and deployed in this web app as the final predictor.

Saved Outputs

The workflow stores the chosen classifier in models/rf_model.pkl and the evaluation summary in models/model_metrics.json, which powers the tabs shown in this interface.

About This Project

NeuroScan is an end-to-end machine learning web application built for IT325 – Machine Learning (2nd Sem SY 2025–2026). It predicts the likelihood of Alzheimer's Disease diagnosis from structured clinical patient data.

Dataset: Gonzales_AlzheimersDisease.csv — 2,149 patient records × 21 features. Target: Diagnosis (0 = No AD, 1 = AD). Class imbalance: 64.6% / 35.4%.

Pipeline: MinMaxScaler → SMOTE → 5 classifiers compared via 5-fold CV → Top 2 (Random Forest, Logistic Regression) tuned with GridSearchCV → Winner selected by F1.

Deployed Model: {{ final_model_name }} — Test F1 Score: {{ winner_f1 }}

Stack: Python · scikit-learn · imbalanced-learn · Flask · HTML/CSS/JS
